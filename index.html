Chi Sono

Ciao! Sono Diego Vincenzi (online Kirito), studente al 4° anno di Informatica all'IIS Copernico Carpeggiani (Ferrara). La tecnologia è la mia più grande passione, dallo sviluppo software alla gestione di infrastrutture.

Gestisco un server Proxmox personale come laboratorio per sperimentare con VM, container, networking e servizi self-hosted. Parallelamente, sono fondatore di GXE Devs e Leader/Sherpa del clan GXE Republic su Destiny 2.

Attualmente sto espandendo le mie competenze nello sviluppo web moderno, lavorando a una dashboard Next.js per il mio clan di Destiny 2. Questo progetto combina la mia passione per il gaming con lo sviluppo di applicazioni web avanzate, utilizzando tecnologie come React, Next.js e Tailwind CSS.

Progetti Principali

GXE Devs - Bot MQL5

Con Artemie Fedoruc, sviluppo bot di trading automatico (MQL5) personalizzati per clienti, con un modello basato su commissioni.

Server Lab Proxmox

Il mio playground personale con Proxmox VE per testare Linux, Docker, servizi di rete e automazione in un ambiente controllato.

GXE Republic - Community

Leader e Sherpa nel clan Destiny 2 GXE Republic. Gestione community, eventi in-game e server Discord. Ottima esperienza di teamwork e leadership.

GXE Dashboard - Next.js

Una moderna dashboard web sviluppata con Next.js per il clan Destiny 2. Include statistiche dei membri, calendario eventi, e gestione dell'attività del clan con integrazione dell'API di Bungie.
